Om oppgåva

Bursdag i Antarktis er ein interaktiv animasjon som fortel historia om ein liten katt som har gått seg bort på bursdagen sin. Heldigvis treff han nokre hyggjelege pingvinar han kan feire saman med.

Dette prosjektet viser fram nokre av moglegheitene Scratch gir for å lage animasjonar og presentasjonar, som samstundes kan koplast opp mot brukarane og gi rom for interaktivitet.

Oppgåva passar til:

Fag: Matematikk, norsk, programmering

Anbefalte trinn: 3.-10. trinn.

Tema: Koordinatsystem, samansett tekst.

Tidsbruk: Dobbelttime eller meir.
